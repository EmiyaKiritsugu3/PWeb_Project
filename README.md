
# Sistema de Gestão para Academia Five Star

Este é um projeto full-stack desenvolvido no Firebase Studio, criando um sistema de gestão completo para uma academia fictícia.

## Visão Geral

A aplicação é dividida em duas experiências principais:

1.  **Painel de Gestão (`/dashboard`):** Uma área administrativa para funcionários gerenciarem alunos, finanças, e planos de treino (inclusive com um gerador via IA).
2.  **Portal do Aluno (`/aluno`):** Uma área de autoatendimento onde os clientes podem ver seus treinos, agendá-los e receber feedback motivacional gerado por IA.

## Tecnologias Utilizadas

*   **Framework:** Next.js (com React e TypeScript)
*   **UI/Estilização:** Tailwind CSS, ShadCN/UI, Lucide React
*   **Backend & Banco de Dados:** Firebase (Firestore e Authentication)
*   **Inteligência Artificial:** Genkit (com modelos Google AI - Gemini)
*   **Formulários:** React Hook Form e Zod
