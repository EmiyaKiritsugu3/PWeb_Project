# Implementation Plan: It3 — AI Workout Integration & E2E Coverage

**Branch**: `feat/005-it3-ai-workouts` | **Date**: 2026-04-18 | **Spec**: `specs/005-it3-ai-workouts/spec.md`

## Summary

Integrate the existing `generateWorkoutFeedback` Genkit flow into `WorkoutSession.tsx` so students
receive AI motivational feedback after completing a workout (US06). Add two P1 E2E test scenarios
that are documented as pending in `CRITICAL-PATHS.md`: workout session completion and student
enrollment. No new flows or schema changes are required.

## Technical Context

**Language/Version**: TypeScript 5 (strict mode) + Next.js 15 App Router  
**Primary Dependencies**: Genkit 1.32 + Gemini 2.5 Flash, Playwright 1.59, Vitest 4  
**Storage**: PostgreSQL via Prisma 7 (no schema changes)  
**Testing**: Vitest (unit) + Playwright (E2E)  
**Target Platform**: Web — Vercel edge + Node.js serverless  
**Performance Goals**: Feedback render < 3s p95 (AI call is async, non-blocking)  
**Constraints**: AI call must degrade gracefully; workout completion must not depend on AI response  
**Scale/Scope**: 3 tasks across 3 files
