# Plano de Iteração e Release - SmartManagementSystem (SMS)

Este documento descreve o cronograma de desenvolvimento para o **SmartManagementSystem (SMS)** durante o semestre 2026.1.

## Plano de Iterações

| Iteração         | Período       | Objetivos                                         | Entregas                           |
| ---------------- | ------------- | ------------------------------------------------- | ---------------------------------- |
| **It0 (Início)** | 10/03 - 26/03 | Planejamento, Estudos e Definição de Tecnologias. | Tarefa 01 (Markdown), Repositório. |
| **It1**          | 27/03 - 17/04 | Documento de Visão, Modelos, User Stories.        | Detalhamento do US00.              |
| **It2**          | 18/04 - 08/05 | Implementação do Dashboard e Gestão de Alunos.    | CRUD de Alunos, Autenticação.      |
| **It3**          | 09/05 - 29/05 | Implementação da Gestão de Treinos e IA.          | Gerador de Treinos com Gemini.     |
| **It4**          | 30/05 - 19/06 | Portal do Aluno e Feedbacks.                      | Portal do Aluno, Gamificação.      |
| **It5**          | 20/06 - 10/07 | Testes Finais e Documentação de Encerramento.     | Sistema Completo (MVP).            |

## Plano de Release

| Release          | Data  | Funcionalidades Incluídas                             |
| ---------------- | ----- | ----------------------------------------------------- |
| **v0.1.0**       | 17/04 | Documentação Inicial e Estrutura de Projeto.          |
| **v0.5.0**       | 08/05 | Sistema de Autenticação e Gestão de Cadastros.        |
| **v1.0.0 (MVP)** | 10/07 | Sistema completo com Dashboard, Portal do Aluno e IA. |
